Cleaning

Extremely RTs

We removed 1162 trials (0.14% of the data) with RTs longer than 6 seconds, as these likely reflect participants taking breaks during the task.

illu <- illu |> filter(RT < 7)

We removed 78 trials (9.69e-03%) where RT equaled 0, as these likely reflect early responses that coincided with stimulus onset.

illu <- illu |> filter(RT > 0)

Extreme Participants

Code 
outliers <- illu |>
  summarize(Error = mean(Error), 
            RT_SD = sd(RT), 
            # RT_Skew = datawizard::skewness(RT),
            # RT_Kurtosis = datawizard::kurtosis(RT),
            RT = mean(RT), 
            .by = c("Participant", "Illusion_Type", "Block", "Dataset")) |> 
  mutate(RT_SD = ifelse(is.na(RT_SD), 1, RT_SD)) |> 
  arrange(desc(Error), desc(RT))

# Rules
outliers$Outlier <- ifelse(outliers$RT < 1/3, "Outlier", "Normal")
outliers$Outlier <- ifelse(outliers$RT > 3, "Outlier", outliers$Outlier)
outliers$Outlier <- ifelse(outliers$Error >= 0.45, "Outlier", outliers$Outlier)
